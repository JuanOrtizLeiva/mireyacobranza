Acerca de este Reporte

Este dashboard muestra la información consolidada de ventas y compras basada en los libros electrónicos de contabilidad. Los datos son extraídos directamente desde el sistema a través de consultas SQL que analizan las siguientes fuentes:

  • Facturas de Venta: Documentos tipo 33 y sus notas de crédito asociadas (tipo 61)
  • Facturas de Compra: Incluye montos netos, exentos e IVA recuperable/no recuperable

Para el cálculo se utilizan las siguientes fórmulas:

  • Ventas Totales: Venta Neta + Exento - Notas de Crédito
  • Compras Totales: Compra Neta + Exento - Notas de Crédito
  • Diferencia: Ventas Totales - Compras Totales
  • Margen Porcentual: (Diferencia / Ventas Totales) × 100
  • Acumulado Anual: Suma progresiva de los valores mensuales para el año seleccionado

Los datos se actualizan en tiempo real al hacer clic en el botón "Actualizar Datos" y provienen directamente de las tablas de la base de datos. Puede utilizar los filtros de año y mes para analizar períodos específicos.
